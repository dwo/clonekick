Clonekick

A Songkick bookmarklet to clone the event you are looking at

Download this project as a .zip file Download this project as a tar.gz file

Writing out the same information over and over again is boring. Let's just clone the event and change the minor details.

Drag this onto your toolbar: clonekick

Click the bookmarklet on any Songkick concert to be taken to a new concert form with artists, venue and date copied in automatically.
